# Learn By Project - Axion

Repository del progetto didattico **Learning by Project**.

Il lavoro simula la presa in carico di una commessa sistemistica per il gruppo alberghiero **Il Sogno**, composto da tre strutture: **La Meraviglia**, **L'Incanto** e **Il Sogno**.

L'obiettivo è progettare e documentare una proposta di infrastruttura IT completa: rete, CED, servizi Windows, Active Directory, DNS, DHCP, sicurezza, gestione utenti, servizi multimediali e budget tecnico.

La repository raccoglie appunti, decisioni tecniche, documentazione operativa, diagrammi, immagini di progetto e materiale di presentazione.
